Apex-SimHub

Este proyecto es nuestra propuesta para la materia y se centra en un tópico que nos apasiona: el simracing. La idea principal es construir una aplicación web integral pensada para pilotos virtuales, agrupando análisis de rendimiento, educación y equipamiento en un solo lugar.

La plataforma va a estar dividida en tres grandes áreas. Primero, un "hub de simuladores" donde se mostrarán juegos como Assetto Corsa Evo, iRacing, Le Mans Ultimate y RBR HU, permitiendo buscar carreras específicas por fecha. Segundo, un "apartado de telemetría y mejora continua", donde los usuarios podrán analizar a fondo sus sesiones. Acá se van a mostrar gráficos del uso de pedales (acelerador, freno, embrague) y marchas, acompañados de un mapa del circuito que dibuja las trazadas reales. Junto a este mapa, habrá una guía de consejos dinámica para ayudar a mejorar los tiempos de vuelta y pulir el setup del coche.

Por último, la web va a incluir una "tienda SimRacer (E-commerce)" de hardware especializado (volantes, cockpits, pedaleras). Está pensada tanto para el que ya sabe lo que busca y quiere ver todo el catálogo libremente, como para el principiante; por eso, al ingresar, la tienda te recibe con un cuestionario opcional que, según tus respuestas, te recomienda tu primera compra ideal o algo un poco mas avanzado.

Todo el ecosistema de la página va a estar desarrollado utilizando HTML, CSS, JavaScript y Angular.
