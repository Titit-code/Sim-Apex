¡Bienvenidos al repositorio de Apex SimHub!

Este proyecto es nuestra propuesta para la materia y se centra en una pasión compartida: el simracing. El objetivo es crear una plataforma web integral que acompañe a los pilotos virtuales, combinando herramientas de análisis, aprendizaje y acceso a equipamiento en un mismo lugar.

La plataforma está organizada en tres grandes áreas: un hub de simuladores, un apartado enfocado en telemetría y mejora de rendimiento, y una tienda especializada en hardware simracing.
